## Этап 3 — Низкий приоритет

### 3.1 Добавить `<noscript>` fallback

В каждый HTML-файл добавить до `</body>`:
```html
<noscript>
  <style>
    .music-toggle, .gallery-grid, #lb-overlay { display: none !important; }
    .noscript-msg { display: block; text-align: center; padding: 2rem; color: #ccc; }
  </style>
  <p class="noscript-msg">Для полной функциональности сайта включите JavaScript.</p>
</noscript>
```

### 3.2 Self-hosting Google Fonts

**Файлы:** все 8 HTML-файлов, `style.css`.
- Скачать используемые шрифты (вероятно, Cormorant + Inter или подобные)
- Разместить в `assets/fonts/`
- Заменить `<link href="https://fonts.googleapis.com/...">` на `@font-face` в `style.css`
- Убрать внешние ссылки на Google Fonts из всех HTML

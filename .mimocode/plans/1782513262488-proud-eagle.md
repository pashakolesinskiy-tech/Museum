# Улучшения дизайна portnova.html

## Текущая структура
- Портрет + стихотворение в две колонки (`.bio-hero-row`)
- Кнопки разделов (награды, архив, книги)
- Модальное окно «Об экспозиции»

## Предложения

### 1. Анимация портрета при загрузке
Добавить плавное появление портрета с эмблемой:
```css
@keyframes portraitReveal {
  from { opacity: 0; transform: scale(0.95); filter: blur(4px); }
  to { opacity: 1; transform: scale(1); filter: blur(0); }
}
.bio-portrait-frame { animation: portraitReveal 1.2s ease-out; }
```
**Файл:** `style.css` — добавить после `.bio-portrait-frame`

### 2. Параллакс-эффект на портрете
При скролле портрет будет медленнее двигаться, создавая глубину:
```js
window.addEventListener("scroll", function() {
  var portrait = document.querySelector(".bio-portrait-frame img");
  if (portrait) {
    var rect = portrait.getBoundingClientRect();
    var offset = (rect.top - window.innerHeight / 2) * 0.08;
    portrait.style.transform = "translateY(" + offset + "px)";
  }
});
```
**Файл:** `script.js` — добавить в конец

### 3. Градиентная линия-разделитель между секциями
Заменить статичную линию на анимированную:
```css
.bio-section-nav::before {
  content: "";
  display: block;
  width: 120px;
  height: 1px;
  background: linear-gradient(90deg, transparent, var(--accent), transparent);
  margin: 0 auto 30px;
}
```
**Файл:** `style.css` — добавить к `.bio-section-nav`

### 4. Кнопки разделов — иконки вместо ✦
Заменить текстовые иконки на SVG:
- Награды → 🏅 (медаль)
- Архив → 📷 (камера)
- Книги → 📖 (книга)

**Файл:** `portnova.html` — строки 236, 243, 248

### 5. Hover-эффект на стихотворении
Добавить подсветку последней строки при наведении на блок:
```css
.bio-poem:hover .bio-poem-last {
  text-shadow: 0 0 20px rgba(212,175,55,0.3);
}
```
**Файл:** `style.css`

### 6. Модальное окно — smoother открытие
Добавить анимацию появления:
```css
.about-modal { opacity: 0; transition: opacity 0.3s; }
.about-modal.active { opacity: 1; }
.about-modal-content { transform: translateY(20px); transition: transform 0.4s; }
.about-modal.active .about-modal-content { transform: translateY(0); }
```
**Файл:** `style.css`

### 7. Мобильная адаптация
На мобильных кнопки разделов в столбик:
```css
@media (max-width: 640px) {
  .bio-section-nav { flex-direction: column; align-items: center; }
  .bio-section-btn { width: 100%; max-width: 320px; }
}
```
**Файл:** `style.css` — в `@media (max-width: 640px)`

## Приоритеты
1. **Высокий:** Анимация портрета, модальное окно
2. **Средний:** Параллакс, кнопки с иконками
3. **Низкий:** Hover-эффекты, мобильная адаптация

## Верификация
1. Открыть `portnova.html` → портрет плавно появляется
2. Навести на стихотворение → подсветка последней строки
3. Кликнуть «Об экспозиции» → модалка плавно открывается
4. Мобильный вид → кнопки в столбик

# Museum Website

Простой адаптивный сайт музея из 3 страниц.

## Технологии

- HTML
- CSS
- Tailwind CSS (упоминание)
- JavaScript

## Страницы

1. Главная — слайдер изображений
2. Выставки — карточки выставок
3. Контакты — форма и карта

## Запуск

1. Скачать проект
2. Открыть файл `index.html` в любом браузере

## 💎 Обновление (Последняя версия)

Проект был полностью переработан для создания премиального визуального опыта:

### Что добавлено и изменено:
- **Премиальная типографика**: Интегрированы шрифты `Cormorant Garamond` (элегантная антиква для заголовков) и `Outfit` (чистый гротеск для читаемости текста).
- **Цветовая палитра**: Дизайн переведен в "дорогой" темный режим (`#050505`) с изящными акцентами цвета золота (`#D4AF37`).
- **Качественные заглушки (Images)**: Подключены высококачественные тематические фотографии с платформы Unsplash с автоматической подгонкой разрешения.
- **Новые векторные иконки**: Базовые SVG заменены на детализированные утонченные иконки в едином стиле (Lucide style) с единой толщиной линий.
- **Glassmorphism**: Добавлен эффект матового стекла в навигационное меню для соответствия современным трендам.
- **Улучшенный UX/Анимации**: 
  - Карточки экспонатов получили градиентное затемнение и эффект плавного увеличения (`scale`) при наведении.
  - Скрипт `IntersectionObserver` переписан: теперь элементы элегантно "всплывают" снизу вверх (`.animate-up`) один раз при первой прокрутке.
  - Улучшена анимация бургер-меню (превращается в крестик).
